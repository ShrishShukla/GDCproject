Technical aspect...

There would be sensors for checking power flow, power quality index and power management. The data collected from these sensors will be used to feed an ML model for water usage predictions and statistics. For this purpose we will use a network where the sensors will act as Z-End devices.

The data collected from these sensors will be sent to Zcoordinator which is connected to nodemcu black wireless. These Beagle board will send the data to a database server using TCP/IP protocol.

The data from database will be fed into ML model for optimum usage
